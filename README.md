# 📌 Weak Degeneracy of Graphs (Using Structural Results)

## 📖 Overview
This project computes the **weak degeneracy** of graphs using known structural results from graph theory instead of brute-force computation.

The program identifies special classes of graphs (such as trees, planar graphs, complete graphs, etc.) and applies theoretical bounds to efficiently determine weak degeneracy.

---

## 🚀 Features

- Detects special graph classes:
  - Complete Graphs
  - Complete Bipartite Graphs
  - Trees
  - Planar Graphs
  - Icosahedral Graph
- Uses theoretical results:
  - Planar graph bounds
  - Girth-based results
  - Forbidden cycle conditions
  - Degree-based bounds (Brooks-type idea)
- Computes:
  - Cycle lengths
  - Forbidden cycle verification
  - Weak degeneracy using best bounds
---

## 🧠 Key Concepts

- Weak Degeneracy
- Graph Coloring
- Planarity
- Girth
- Forbidden Subgraphs
- Structural Graph Theory
